Реализованно приложение, которое хранит и управляет данными о студентах. 
Данные поступают ввиде csv файлов.
Формат csv файла:
Фамилия,Имя,Отчество,Дата рождения
Для хранения информации в базе данных (MySQL) исползуются таблицы Group и Student
Есть возможность вывода всех студентов на экран, сгруппированных по группам. 
Например
Группа 17704
Иванов Иван Иванович 1999-06-01
Петров Петр Петрович 2000-01-21

Команда запуска:
> java -jar student-manager.jar updateDB c:/someFolder/studentDir/
Где c:/someFolder/studentDir/
